ДОМАШНЕЕ ЗАДАНИЕ #12
------------------------------------
Пользователь вводит пример такого вида:
2 + 2 - 2
Результат программы:
2

*Реализовать правила математики:
2 + 2 * 2
Результат программы:
6

**Реализовать скобочки
( 2 + 2 ) * 2
Результат программы:
8


ДОБАВИТЬ ИСКЛЮЧЕНИЯ:
NumberFormatException
ArithmeticException
